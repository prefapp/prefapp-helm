Prefapp Helm

Prefapp Helm is a helm microframework for implementing modular charts and umbrella charts.

Checkout all the capabilities!

Renders

Avoidance of hardcoding the artifact structure with the possibility of embedding block sections of the values directly on the data section. Less code equals ease of debug.

Open Source

Updated and tested daily. No hidden magic over here: open to the public!

Data blocks

The "ph.override" method allows a data block to override another by merging both, giving preference to the first one.
